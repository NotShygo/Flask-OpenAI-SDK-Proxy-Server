# LLM Proxy Server & Client

A simple proxy server that forwards chat completion requests to an LLM API (OpenAI-compatible) and a Python client module to interact with it.

## Features

- **Proxy Server**: Accepts POST requests at `/chat`, forwards them to a configured LLM endpoint, and returns the response.
- **Client Module**: Provides functions to send text-only or multimodal (text + image) messages to the proxy.
- Supports both text and image inputs (using base64‑encoded images).

## Requirements

- Python 3.9+
- Flask
- OpenAI Python library (for the server)
- (Optional) environment variables for configuration

## Installation

```bash
git clone https://github.com/NotShygo/Flask-OpenAI-SDK-Proxy-Server
cd Flask-OpenAI-SDK-Proxy-Server
pip install flask openai
```

## Configuration

### Proxy Server (`proxy_server.py`)

Edit the following lines in `proxy_server.py`:

```python
client = OpenAI(
    base_url="BASE_URL",   # e.g., "https://api.openai.com/v1"
    api_key="API_KEY"      # your API key
)
```

### Client (`example_request.py`)

Set the server URL and model name either by editing the file:

```python
SERVER_URL = "SERVER_URL"  # replace with actual IP generated by proxy_server.py
```

## Usage

### Start the Proxy Server

```bash
python proxy_server.py
```

The server will run on `http://0.0.0.0:5000` (by default). Ensure the port is accessible if running remotely.

### Use the Client

Import the client functions in your own script or run the example:

```python
from example_request import text_only, text_with_image

# Text-only request
response = text_only("What is the capital of France?")
print(response)

# Request with an image
response = text_with_image("path/to/image.jpg", "Is there a dog in this picture?")
print(response)
```

#### Available Functions

- `call_llm(messages, model)`: Low-level function that sends any list of messages (following the OpenAI Chat Completions format) to the proxy.
- `text_only(obj)`: Sends a plain text prompt (replace `YOUR_PROMPT` with your actual prompt).
- `text_with_image(image_path, obj)`: Sends a prompt together with a base64‑encoded image (the prompt should be set inside the function).

**Note**: The placeholder prompts `YOUR_PROMPT` in the client functions should be replaced with your actual instructions.

## File Structure

- `proxy_server.py`: Flask app that receives messages and forwards them to the LLM endpoint.
- `example_request.py`: Client module with helper functions to call the proxy.
